# Subsystem Overview

<div></div>
### What is a Subsystem?

An FRC robot is is made from multiple <u>Subsystems</u>, or parts of a robot that each perform a task seperately and while working together. 
<div></div>

### How do we program subsystems?

We use Java object-oriented programming to program our robot. We create subsystems using a class and give it functions using commands.
